语言
Kotlin New
现代 简洁 安全 兼容Java 被赞誉为Android世界里的Swift。 Kotlin还可以编译成Java字节码
Java
一次编写，到处运行的特点 功能强大 Java语言最大的优点就是与平台无关。
Python
干净 简单易用 功能强大 标准库和第三库众多，简单就是它的最大魅力
C
代码量小 运行速度快 它兼顾了高级语言和汇编语言的优点,相较于其它编程语言具有较大优势
C++
信息隐藏 多态 继承 C++ 可运行于多种平台上，如 Windows、MAC、以及 UNIX 的各种版本
C#
安全 稳定 优雅 简单 它继承C和C++强大功能同时去掉了一些它们的复杂特性
Go
效率 性能 安全 健壮 Go与C相近,但功能上有:内存安全,GC(垃圾回收)
PHP
免费的 脚本语言 交互 是在服务器端执行的脚本语言,尤其适用于Web开发并可嵌入HTML中

前端
HTML
超文本标记语言 它通过一系列标签可以将分散的Internet资源连接为一个逻辑整体
HTML5 New
HTML5是下一代HTML 标准 但仍处于完善之中,然而大部分现代浏览器已经具备了某HTML5支持
CSS
层叠样式表 是一种用来表现HTML或XML等文件样式的计算机语言。对网页各元素进行格式化
CSS3 New
层叠样式表 css3作为css进阶版,拆分和增加了盒子模型、列表模块、语言模块 、背景边框等

框架
BootStrap 前端
前端组件库 响应式布局 有了BootStrap之后,安卓电脑可以只做一个项目
Vue 前端
易用 灵活 高效 是一套用于构建用户界面的渐进式框架。Vue 被设计为可以自底向上逐层应用
Spring 后端
轻量级开源框架 它为容易而快速的开发出耐用的 Java 应用程序提供了全面的基础设施
Spring MVC 后端
高度可配置 SpringMVC提供了模型-视图-控制体系结构,可以用来开发web应用程序的组件
MyBatis 后端
持久层框架 MyBatis 免除了几乎所有的 JDBC 代码以及设置参数和获取结果集的工作
Hibernate 后端
对象关系映射框架 开放源代码 它作为数据库与界面之间的桥梁,需要面向对象思想操纵对象
Struts 2 后端
简洁 可扩展 Struts2是一个基于MVC设计模式的Web应用框架，它本质上相当于一个servlet
